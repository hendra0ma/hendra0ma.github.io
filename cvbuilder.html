Panduan Cetak / Ekspor PDF

Untuk hasil CV terbaik dengan tata letak profesional, harap atur parameter berikut di dialog cetak browser Anda:

  • 1. Pilih Tujuan (**Destination**): **Simpan sebagai PDF (Save as PDF)**.
  • 2. Ukuran Kertas (**Paper Size**): Pastikan disetel ke **A4**.
  • 3. Margin: Disarankan disetel ke **Tidak Ada (None)** atau **Default**.
  • 4. Opsi Tambahan: **Centang** pilihan **Grafik Latar Belakang (Background Graphics)** agar elemen warna CV muncul sempurna.
